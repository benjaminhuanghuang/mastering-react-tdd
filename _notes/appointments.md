## Setup
- Install jest
```
  npm i -D jest
```

- Install react
```
  npm i -D react react-dom
```
- Install and setup Bable
```
  npm i -D @babel/preset-env @babel/preset-react
  npm i -D @babel/plugin-transform-runtime
  npm i -S @babel/runtime
```
